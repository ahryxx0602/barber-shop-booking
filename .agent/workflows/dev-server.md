## Khởi động dev server

// turbo-all

1. Chạy Laravel dev server:
```bash
cd /home/vanthanh/Ind-Project/Laravel/barbershop && php artisan serve --host=0.0.0.0 --port=8000
```

2. Mở terminal mới, chạy Vite:
```bash
cd /home/vanthanh/Ind-Project/Laravel/barbershop && npm run dev
```

3. Thông báo cho user rằng server đã sẵn sàng tại `http://localhost:8000`.
